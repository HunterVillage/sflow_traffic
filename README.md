# sflow_traffic
python解析sflow报文并存储到mysql

## 说明
sflowtool version: 5.02
Python version 2.7.5

1.sflowtool将sflow报文转为netsFlow报文并发到本机指定端口（udp）    
2.抓包程序获取指定端口的数据，根据ip+port获取服务之间的调用关系，并存储到mysql

## 启动
> sh start.sh
